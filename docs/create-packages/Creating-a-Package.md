### <a name="additional-options"></a><span data-ttu-id="531fb-302">追加オプション</span><span class="sxs-lookup"><span data-stu-id="531fb-302">Additional options</span></span>

- <span data-ttu-id="531fb-312">**シンボル**: 利用者がデバッガーでパッケージ コードをステップ実行することを可能にするシンボルを含めるには、`-Symbols` オプションを使用します。</span><span class="sxs-lookup"><span data-stu-id="531fb-312">**Symbols**: to include symbols that allow consumers to step through your package code in the debugger, use the `-Symbols` option:</span></span>
