## <a name="matching-assembly-versions-and-the-target-framework-in-a-project"></a><span data-ttu-id="a7f77-129">プロジェクトでアセンブリ バージョンとターゲット フレームワークを組み合わせる</span><span class="sxs-lookup"><span data-stu-id="a7f77-129">Matching assembly versions and the target framework in a project</span></span>
