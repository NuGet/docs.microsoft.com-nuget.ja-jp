# <a name="nuget-error-nu5041"></a>NuGet エラー NU5041

<pre>The readme file 'readme.md' is empty.</pre>


### <a name="issue"></a>問題 

パッケージ readme として指定されたファイルが空です。


### <a name="solution"></a>解決策

Readme ファイルの内容を入力します。
