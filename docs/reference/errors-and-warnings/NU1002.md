# <a name="nuget-error-nu1002"></a>NuGet エラー NU1002

<pre>'CLEAR' cannot be used in conjunction with other values</pre>

### <a name="issue"></a>懸案事項
入力とクリア キーワードの組み合わせが無効です。

### <a name="solution"></a>ソリューション
クリアを単独で使用し、その他のすべての入力を省略します。
